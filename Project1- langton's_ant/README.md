## Langton's Ant

A Simulation of Langton's Ant which you can learn about at this [Wikipedia link](https://en.wikipedia.org/wiki/Langton%27s_ant). The simulation can be run by downloading the zip file and unzipping it. Then type make on the command line and then type langtonAnt to start the simulation and follow the menu prompts.
